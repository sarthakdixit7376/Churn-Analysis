# 📊 Customer Churn Prediction System

## 🚀 Overview
This project focuses on building an end-to-end customer churn prediction pipeline using SQL, Power BI, and Machine Learning. The goal is to identify customers who are likely to churn and provide actionable insights to improve retention strategies.

---

## 🎯 Problem Statement
Customer churn is a critical problem for businesses, especially in telecom and subscription-based industries. Retaining existing customers is more cost-effective than acquiring new ones. This project aims to predict churn and analyze key factors influencing it.

---

## 🛠️ Tech Stack
- **SQL** – Data extraction, transformation (ETL)
- **Power BI** – Data visualization and dashboarding
- **Python (Scikit-learn, Pandas, NumPy)** – Machine Learning model building

---

## ⚙️ Project Pipeline

### 1. Data Extraction & ETL (SQL)
- Cleaned raw data (handled missing values, duplicates)
- Performed transformations and feature engineering
- Used joins and aggregations to create meaningful datasets

### 2. Data Visualization (Power BI)
- Built interactive dashboards to analyze churn trends
- Key KPIs:
  - Churn Rate
  - Customer Segmentation
  - Revenue Impact
- Identified patterns like:
  - Higher churn in low-tenure customers
  - Increased churn with higher monthly charges

### 3. Machine Learning Model
- Applied classification algorithms to predict churn
- Models used:
  - Logistic Regression
- Evaluation Metrics:
  - Accuracy
  - Precision
  - Recall
  - ROC-AUC Score

> ⚠️ Focus was given to **Recall**, as missing a churn-prone customer is more costly for the business.

---

## 📊 Results
- Improved recall for churn prediction
- Identified key features influencing churn:
  - Tenure
  - Monthly Charges
  - Contract Type

---

## 💡 Key Insights
- Customers with shorter tenure are more likely to churn
- High monthly charges correlate with higher churn
- Contract type significantly impacts retention

---

## 📈 Business Recommendations
- Offer incentives to high-risk customers
- Improve onboarding experience for new users
- Introduce flexible pricing plans
